# 🦆 Goose Game Super

**Goose Game Super** — аркадна гра на Python з Pygame, де ви керуєте гускою, збираєте бонуси та уникаєте ворогів. Гра має динамічний фон, рівні складності та музику.  

⚡ **Створена за підтримки [GO-IT](https://go-it.global/)**

---

## 🎮 Керування
- Стрілки ← → ↑ ↓ — рух гуски  
- **SPACE** — пауза / продовження  
- **ENTER** — рестарт після поразки  
- **ESC** — вихід з гри  

---

## 🛠️ Як запустити
1. Створіть та активуйте віртуальне середовище:
```bash
python3 -m venv .venv
source .venv/bin/activate    # Linux/macOS
.venv\Scripts\activate       # Windows

Встановіть Pygame:

pip install pygame

Запуск гри:

python main.py

Якщо деякі файли (background.png, player.png, enemy.png, bonus.png, music.mp3) відсутні, гра автоматично створює базові графічні об’єкти і працює без музики.

🎨 Особливості

Скролінг фону

Вороги та бонуси, які з’являються випадково

Рівні складності, які підвищують швидкість та частоту появи ворогів

Анімація гравця (якщо в папці Goose є кілька зображень)

HUD з очками, рівнем і прогресом до наступного рівня

📂 Структура проекту
goose_game_super/
├── main.py           # Головний скрипт гри
├── music.mp3         # Фонове аудіо
├── background.png    # Фонове зображення
├── player.png        # Зображення гуски (якщо є)
├── enemy.png         # Зображення ворогів
├── bonus.png         # Зображення бонусів
├── Goose/            # Папка зі спрайтами гравця (може містити кілька картинок)
├── README.md         # Цей файл
├── venv/             # Віртуальне середовище
└── build/ dist/      # Автогенеровані файли (для PyInstaller)
